About Emoji Cipher


Emoji Cipher is an online text encryption program. Through this site, you can encode any texts into emoji symbols for security purposes such as encryption, obfuscation etc. If you are new to text encryption, you can also try it for fun.

When a user tries to encrypt a piece of text, the program generates a random key and applies the key into the AES Encryption Algorithm to get the encrypted ciphertext. Then the key and the ciphertext are combined together and translated to emoji symbols. For the same text input, different keys would outcome different ciphertexts. Therefore users would see different results when they try to encrypt the same text repeatedly.

This site won't store any of your personal data such as text input/output, ip address etc.

This program is open-source and can be found in Github.
